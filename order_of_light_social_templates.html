Recurring Layer Effects — Save as Adobe Styles
Grain Overlay
Filter > Noise > Add Noise · 3% · Gaussian · Monochromatic · Overlay blend mode · 60% opacity
Scan Lines
Pattern fill · 1px black line every 4px · Multiply blend · 8% opacity · on top of everything
Vignette
Radial gradient fill layer · Black edges → transparent centre · Multiply · 70% opacity
Photo Colour Grade
Hue/Sat: -30 saturation · Curves: push shadows to #1A0A0A · Add warm amber tone via Colour Balance (shadows: +R+8, +Y+6)
Mark SVG Import
Place as Smart Object · Link to master SVG · Recolour via Layer Style Colour Overlay for each variant
